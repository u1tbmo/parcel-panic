# Parcel Panic 📦

A fast-paced, top-down multiplayer delivery game.

## Prerequisites
- **Java 25**: This project requires JDK 25.

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/u1tbmo/parcel-panic.git
    cd parcel-panic
    ```

2.  **Configure Local Environment**:
    The project is configured to look for the JDK or install it if a compatible JDK can't be found.

3.  **Run the Game**:
    Use the included Gradle wrapper:
    ```bash
    ./gradlew run
    ```
